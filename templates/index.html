How to use

Encryption

Enter your desired input message into the "Enter message" box, and select encryption as your mode. Once you get your output, you will also receive a message ID which is needed for decryption. Make sure to store that ID or else decryption is impossible!

Decryption

Enter your desired input message into the "Enter message" box, select decryption as your mode, and enter your message ID which you should've received when you first encrypted the message. It is not possible to decrypt without a message ID!
